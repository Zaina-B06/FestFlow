# 🎉 FestFlow-AI Event Management System

FestFlow is an AI-powered platform that helps manage college events by intelligently coordinating volunteers, tasks, and real-time operations.

---

## 🚀 Problem
Event coordination is chaotic — scattered communication, poor task allocation, and no real-time control during emergencies.

---

## 💡 Solution
FestFlow acts as an AI agent that:
- Assigns volunteers based on skill, availability, and location  
- Detects dropouts and reassigns instantly  
- Handles crowd surges with dynamic resource allocation  
- Provides real-time visibility to organizers and volunteers  

---

## 🧠 Key Features
- Smart Volunteer Assignment  
- Auto Reassignment (Dropout Handling)  
- 🚨 Emergency Boost (Crowd Surge)  
- 🗺️ Zone-Based Heatmap  
- Real-Time Notifications  
- Task & Schedule Management  

---

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JS  
- Backend: FastAPI (Python)  
- AI (optional): Groq (LLaMA3)  
- Integration (planned): WhatsApp API
